- **BUG修复**: `start-server.bat`、`restart-server.bat`、`start.bat` 运行时出现 `'姝ｅ湪鍚姩' 不是内部或外部命令` 乱码错误。根因：文件为 UTF-8 without BOM，Windows cmd 默认按 GBK 解析 UTF-8 中文导致命令解析失败。修复：3 个 bat 文件全部转换为 UTF-8 with BOM 编码（EF BB BF），Windows 能正确识别 UTF-8 并配合 `chcp 65001` 正常显示中文。

- **Leaf 数量动态化**: js/layout-engine.js 中 Leaf 对数从硬编码 LAYOUT.leafPairsPerPod(8) 改为根据硬件计算结果动态计算：`leafPairsPerPod = Math.ceil(cm.leafSwitches / podCount / 2)`。POD 子标签从硬编码"包含 16台 Leaf, 下联 64台 算力节点"改为动态文本，显示实际 Leaf 台数和服务器台数

- **BUG修复**: 拓扑图和统计信息完全无法显示。根因：js/layout-engine.js 中 `actualServersInPod` 变量在同一 for 循环作用域内被重复声明了两次（第159行新增 + 第232行原有），`const` 重复声明导致 `SyntaxError: Identifier 'actualServersInPod' has already been declared`。整个脚本因语法错误无法执行。修复：删除第159行的 `const actualServersInPod` 声明，直接在模板字符串中使用 `Math.min(...)` 表达式。重新构建后语法检查通过，预览正常。
